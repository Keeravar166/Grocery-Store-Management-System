# Grocery-Store-Management-System
SQL project for managing grocery store sales, customers, products, and inventory.

## Project Overview
Developed a Grocery Store Management System using MySQL.

## Database Tables
- Customers
- Products
- Orders
- Order Details
- Suppliers
- Employees
- Categories

## SQL Concepts Used
- Joins
- Aggregate Functions
- Group By
- Subqueries
- Views
- Stored Procedures

## Business Questions Answered
1. Top-selling products
2. Customer purchase behavior
3. Supplier contribution
4. Employee performance
5. Monthly sales trends

## Tools Used
- MySQL
- SQL
